## 📸 Overview

Santo Tech is a full-cycle product design & development company website featuring:

- ✅ Modern enterprise SaaS design system
- ✅ Blue `#2563EB` → Cyan `#06B6D4` gradient branding
- ✅ Dark navy `#0F172A` as secondary color
- ✅ Fully responsive across all screen sizes
- ✅ Smooth animations & micro-interactions
- ✅ SEO-optimized HTML structure

---

## ⚡ Features

- **Hexagon background pattern** — subtle animated hex grid with parallax on mouse move
- **Scroll-reveal animations** — cards fade in as you scroll into view
- **Active nav highlighting** — current section highlighted via IntersectionObserver
- **Mobile hamburger** — animated 3-bar → X transition
- **Contact form** — success animation with auto-reset
- **Hover micro-interactions** — lift effects on cards, pills, and buttons
- **Glassmorphism** — backdrop-blur on navbar and case study cards

## 🚀 Getting Started

### Run Locally (No build step needed)

```bash
# Option 1 — npx serve (recommended)
npx serve . -p 3000

# Option 2 — Python
python -m http.server 3000

# Option 3 — Just open the file
start index.html
```

Then open **http://localhost:3000** in your browser.
